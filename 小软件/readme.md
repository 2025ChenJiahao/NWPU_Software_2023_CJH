# 开源小软件集合

## 📖 仓库介绍

本仓库用于开源本人开发的各类小软件、工具脚本及轻量应用，涵盖日常实用工具、学习练手项目、场景化小工具等，无复杂依赖，注重简洁、高效、易用。

所有软件均为个人独立开发，开源目的在于分享技术实现、接受社区反馈，同时也希望能为有同类需求的用户提供便利，欢迎大家 Star、Fork、提 Issue 和 PR，一起完善这些小工具。

## 📦 包含的软件类型（持续更新）

- **日常实用工具**：如文件处理、格式转换、快捷操作类小软件，解决日常办公/生活中的小痛点；
- **学习练手项目**：基于不同技术栈开发的demo级小软件，适合新手参考学习；
- **场景化小工具**：针对特定场景（如数据统计、简单爬虫、本地服务）开发的轻量应用；
- **脚本工具**：各类自动化脚本、批量处理脚本，提升效率。

## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/你的用户名/仓库名.git
cd 仓库名
```

### 2. 查看具体软件

每个软件均放在独立目录下，目录名称对应软件名称，每个目录内包含该软件的详细说明（README.md）、源码及运行/编译说明，请进入对应目录查看具体使用方法。

## 🔧 开发与贡献

### 贡献指南

1. Fork 本仓库；
2. 创建 feature 分支（`git checkout -b feature/xxx`）；
3. 提交代码（`git commit -m 'add: 新增xxx功能'`）；
4. 推送至分支（`git push origin feature/xxx`）；
5. 打开 Pull Request，说明修改内容及用途。

### 反馈问题

若使用过程中遇到 BUG、有功能建议，或对源码有疑问，欢迎在 Issues 中提交，我会尽快回复处理。
