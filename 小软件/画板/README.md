# Pixel Art Editor

一个纯前端的像素画编辑器，单个 HTML 文件即可运行，无需任何依赖。

## 功能

- **绘图工具**：画笔、橡皮擦、油漆桶（泛洪填充）、吸管、直线、矩形、圆形
- **颜色面板**：72 色预设调色板 + 自定义取色器 + 透明色
- **画布设置**：支持 4~128 像素的自定义尺寸，提供 8/16/32/64 快捷预设
- **网格显示**：可开关的辅助网格
- **历史记录**：撤销 / 重做（最多 100 步），支持一键撤销全部
- **导出**：导出原始尺寸 PNG 或 4 倍放大 PNG
- **视图控制**：Alt+滚轮缩放，鼠标中键拖拽平移
- **触摸支持**：适配触屏设备
- **陪伴精灵**：一个像素风小精灵，会在屏幕上闲逛、说话、眨眼睛，可拖拽移动，右键菜单还能打开小游戏
- **恐龙快跑**：内置一个像素风跑酷小游戏，空格或点击跳跃

## 快捷键

| 按键 | 功能 |
|------|------|
| `B` | 画笔 |
| `E` | 橡皮擦 |
| `G` | 油漆桶 |
| `I` | 吸管 |
| `L` | 直线 |
| `R` | 矩形 |
| `C` | 圆形 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |

## 使用方式

直接在浏览器中打开 `pixel-art-editor.html` 即可。
